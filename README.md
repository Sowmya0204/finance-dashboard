# Finance Dashboard UI

A clean, interactive finance dashboard built with React, Tailwind CSS, and Recharts.

## Features
- **Dashboard Overview:** Visual summary of balance, income, and expenses.
- **Data Visualization:** Categorical spending breakdown using Recharts.
- **Transaction Management:** Searchable list of all financial activities.
- **Simulated RBAC:** Toggle between 'Viewer' and 'Admin' roles to see UI changes (Admin can see action buttons).
- **Responsive Design:** Fully functional across mobile, tablet, and desktop.

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **State Management:** React Context API

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
